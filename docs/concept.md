# Concept

## Why

多くの意思決定や思考は、個人の頭の中に閉じている。
その結果、再現性がなく、他者に共有できず、組織として蓄積されない。

## What

人の思考や意思決定を分解し、再利用可能な形で扱う。
思考を「コードのように」構造化・再利用・組み合わせ可能にする。

これを「Mind as Code」と呼ぶ。

## What's Different

単なるプロンプト集ではない。

- 思考（Thinking）
- 役割（Mode）
- 表現（Output）
- 受け手（Receiver）

を分離し、組み合わせ可能にした点に特徴がある。

## Value

- 思考の再現性が高まる
- 組織で意思決定を共有できる
- 個人依存を減らすことができる
- 思考を改善・進化させることができる

## How to Use

以下を組み合わせて利用する。

| レイヤー | 役割 |
|---|---|
| core | 思考原則 |
| thinking-mode | 役割 |
| output-mode | 表現 |
| receiver-model | 受け手の状態 |
| usecase | 具体的なシナリオ |

状況に応じた組み合わせにより、意思決定を再現できる。

## Stance

これは「正解」を提供するものではない。
どう考えるか、どう判断するかを外部化し、改善可能にするための仕組みである。
