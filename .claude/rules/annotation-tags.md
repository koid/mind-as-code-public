# Annotation Tags

core の原則に付与する annotation の許可セットを定義する。

定義の詳細は [core/README.md](../../core/README.md) の Annotation Tags セクションを参照。

## 許可タグ

| tag | meaning |
|---|---|
| [補正] | 素の傾向では逆方向に流れがちな場面で、意識的に押し戻す原則 |

`[補正]` 以外のタグ（例: `[素]` / `[強化]` / `[警戒]`）は使用しない。

## 検証ルール

core/ 配下の原則ファイルにおいて、リスト項目（`- ` または `N. ` で始まる行）の先頭に付与される annotation は `[補正]` のみとする。

それ以外のタグが検出された場合は ERROR。

## 適用範囲

- 対象: core/*.md のリスト項目の先頭タグ
- 対象外: テーブル行、Markdownリンク（`[name.md](path)`）、見出し
- コードブロック内の例示も検査対象に含まれる（定義と例示の一貫性を担保するため）
