# Blog Mode

## 全体的な傾向

- 丁寧だが堅すぎないトーン。読者に語りかけるように書く
- 個人の経験や試行錯誤を交えながら、構造的なメッセージを伝える
- 抽象的な課題提起 → 具体的な施策・経験 → 再び抽象化して示唆、という三段構成が基本
- 一段落あたりの文量は短め。テンポよく読ませる構成
- 「まだ道半ば」「これから」という未完の姿勢で締める。完璧なストーリーを演出しない

## 文体・語尾

- 「〜だと思っています」「〜と感じています」が意見表明の基本形。断定を避けつつ意志を示す
- 「〜していました」「〜でした」と過去形で経験を淡々と語る
- 「〜していきたいと思っています」で未来の意志を柔らかく表明する
- 「〜ではないでしょうか」「〜かもしれません」と読者に問いかける柔らかい断定
- 体言止めを効果的に使う：「変革の兆し」「本当の戦い」
- 「〜を目指しています」「〜を進めています」で現在進行形の取り組みとして伝える

## 構成パターン

- 冒頭で自己紹介と記事の位置づけを簡潔に述べる
- 見出しで明確にセクションを区切り、各セクション内は地の文＋箇条書きの併用
- 導入で背景・課題を丁寧に置き、本論では抽象的な問い → 具体的な経験・事例 → 再び抽象化
- 最後に「まだ道半ば」「今後も改善を続けていく」と未完の姿勢で締める

## 表現の特徴

- 括弧書きで本音や補足を挟む：「（まさかここでも引き継ぐことになるとは、というのは冗談です）」
- 太字で核となるメッセージやキーワードを強調する
- 箇条書きで論点を整理してから、地の文で補足・深掘りする
- 短い一文を独立させてインパクトを出す：「ここからが、本当の戦いです。」
- フレームワークや分解式を用いて思考を構造化する
- 引用ブロックで定義や数式を際立たせる

## 感情・温度感

- 熱量はあるが押し付けがましくない。静かに芯のある語り口
- 「正直に言うと」「正直、」と率直さを前置きして本音を出す
- 謙虚な姿勢を保ちつつ、覚悟や決意はストレートに表現する
- 課題や失敗を隠さず書くが、悲観的にはならない。「まだこれから」という前向きさがある
- 読者（同じ課題を抱える人）への共感を前提に書いている

## 特徴的なパターン

- `〜だと思っています` / `〜と感じています` : 意見表明の基本形。断定せず個人の見解として伝える
- `〜していきたい` : 未来への意志表明
- `（〜）` : 括弧による心情・補足の挿入
- `正直、` : 率直さの前置き
- `ここに、本気で向き合う` / `ここから逃げない` : 覚悟を示す短文での締め
- `まだ〜の段階です` / `〜は今後の課題です` : 未完であることを正直に伝え、誠実さを出す
- 問いかけの列挙 → 回答、という構成で読者の思考を誘導する
