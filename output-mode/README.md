# Output Mode — 表現（伝え方）

出力チャネル別の文体・トーンを定義するレイヤー。
[base.md](base.md) の基本スタイルをベースとし、各チャネルの特性に応じて重ねて適用する。

## 位置づけ

output-mode は「どう伝えるか」を決める。
同じ思考・同じ視点でも、伝え方を変えることで異なるチャネルに適応できる。

thinking-mode、receiver-model とは独立しており、自由に組み合わせ可能。

## 構成

| ファイル | チャネル | 概要 |
|---|---|---|
| [base.md](base.md) | 共通 | 全チャネル共通の基本出力スタイル |
| [blog.md](blog.md) | ブログ記事 | 読者に語りかける丁寧だが堅すぎないトーン |
| [presentation.md](presentation.md) | プレゼンテーション | スライド＋口頭の発表スタイル |

## 拡張ルール

新しい表現を追加する場合は、このレイヤーにファイルを追加する。
core の原則、thinking-mode の視点、receiver-model の受け手定義をここに混ぜないこと。
